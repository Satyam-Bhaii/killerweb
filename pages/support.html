Frequently Asked Questions

My server won't start ‚Äî what do I do?+

Check the console for error messages. Common issues include incorrect Java flags, missing EULA agreement, or corrupted world files. Our support team can assist if needed.

How do I increase RAM on my server?+

You can upgrade your plan directly from the billing panel. After upgrading, the new RAM is allocated automatically on the next restart.

Can I install custom JAR files?+

Yes! Upload any compatible server JAR via the File Manager and select it in the Startup settings.
